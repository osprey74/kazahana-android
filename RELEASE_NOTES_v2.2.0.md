# v2.2.0

## English

- Support Bluesky's new 2MB image upload limit with automatic fallback for legacy PDSes.
- Improved image downsizing to match the official Bluesky app (better quality for large photos).
- Multi-image posts now use a swipeable carousel with page indicators, replacing the old grid layout.

## 日本語

- Bluesky の新しい 2MB 画像アップロード上限に対応（旧 PDS 向けの自動フォールバックあり）。
- 大きな写真の画質劣化を抑える、公式アプリ互換の段階的ダウンサイジングを実装。
- 複数画像投稿の表示を、スワイプ可能なカルーセル（ページインジケーター付き）に変更。
