# v3.0.0

## English

- **Standard Site extended link cards.** Links from Standard Site publishers (Leaflet, pckt, Offprint, etc.) now show a rich card with the publication name, author, publish date, reading time, and the publisher's accent color — matching the official Bluesky app.
- **Posting Standard Site cards.** When you paste a Standard Site link in the composer, the extended preview is fetched via the AppView and the document/publication references are attached to your post, so viewers on any client see the full card.
- Normal OGP link cards are unchanged.

## 日本語

- **Standard Site 拡張リンクカードに対応しました。** Standard Site 採用サービス（Leaflet / pckt / Offprint など）のリンクが、パブリケーション名・著者・公開日・読了時間・テーマカラーを含むリッチなカードで表示されます（Bluesky 公式アプリ準拠）。
- **Standard Site カードの投稿に対応しました。** コンポーザーに Standard Site のリンクを貼り付けると、AppView 経由で拡張プレビューを取得し、ドキュメント／パブリケーションの参照情報を投稿に付与します。どのクライアントの閲覧者にも拡張カードが表示されます。
- 通常の OGP リンクカードの表示はこれまでどおりです。
