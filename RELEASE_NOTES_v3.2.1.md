# v3.2.1

## English

- **Fixed: the "New message" button on the Messages screen is now always visible.** It has moved to the top bar (matching the iOS version), so you can start a new conversation even while an evacuation alert banner is shown. Previously the button could be hidden behind the banner.
- **The compose (new post) button now appears only on the Home and Profile tabs.** It has been removed from screens where it was not needed, such as Search, Notifications, and detail screens.
- **The evacuation alert banner is no longer shown on the Settings screen or the shelter / compass navigation screens,** where it was redundant.

## 日本語

- **メッセージ画面の「新規メッセージ」ボタンが常に表示されるようになりました（不具合修正）。** iOS 版に合わせて画面上部のバーに移動したため、避難情報バナーの表示中でも新しい会話を始められます。これまではバナーにボタンが隠れてしまう場合がありました。
- **新規投稿ボタンをホーム・プロフィールタブのみに表示するようにしました。** 検索・通知・各種詳細画面など、必要のない画面からは表示しないようにしました。
- **避難情報バナーを、設定画面および避難所一覧・簡易ナビ（コンパス）画面では表示しないようにしました。** これらの画面では冗長だったためです。
