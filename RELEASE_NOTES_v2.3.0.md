# v2.3.0

## English

- Added support for long-form post services (standard.site-compatible). Set the posting URL in Settings to show a "Write long-form" shortcut in the composer.
- Localized previously English-only notification text and accessibility labels; translated the Starter Packs label in Spanish and French.
- Settings now shows the actual installed app version (was stale).

## 日本語

- 長文投稿サービス (standard.site 互換) に対応。設定で投稿ページの URL を登録すると、コンポーザーから「長文を書く」ショートカットが使えるようになります。
- 通知の本文や読み上げ用ラベルなど、英語のままだった文言をローカライズ。スターターパックの表示をスペイン語・フランス語にも翻訳。
- 設定画面のバージョン表示が実際のインストール版を反映するようになりました（以前は古い値で固定）。
